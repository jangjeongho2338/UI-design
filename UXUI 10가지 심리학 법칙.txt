UX/UI 10가지 심리학 법칙

1. 제이콥의 법칙

	- 제이콥이 스스로 인간 본성의 법칙이라고 한 법칙
	- 사용자가 다른 웹사이트를 통한 경험을 바탕으로 디자인관례에 대한 기대치를 형성하는 경향을 보임
	- 디자이너들이 일반적인 디자인 관습을 따를 것을 권장
	- 특이한 인터페이스 사용시 사용자의 불만, 혼란 야기 가능 > 서비스 이탈률 증가로 이어짐
	- 멘탈모델과도 관련 있음 (우리가 어떤 시스템에 관해 특히 그 시스템의 작동 방식에 관해 알고 있다고 생각하는 것)

제이콥의 법칙 활용 팁
	- 익숙한 UI 패턴 활용
	- 로그인 방식, 버튼 위치, 인터페이스는 사용자가 아는 구조를 그대로 사용
	- 차별화는 기능이 아닌 감성부터
	- 구조나 흐름은 익숙하게 하고, 컬러나 애니메이션, 마이크로 인터랙션으로 감성적인 차별화 시도
	- 새로운 방식은 힌트를 함께 (슬라이드를 유도하는 화살표, 클릭가능한 UI만 표시등)
	- 좋은 UX는 신기보단 편하다에서 시작됨

2. 힉의 법칙

	- 선택 가능한 옵션 수가 증가할수록 의사 결정 시간이 기하급수적으로 증가한다는 개념 기반
	- 심리학 및 인간-컴퓨터의 상호작용 분야에서 중요한 원칙
	- RT=a+b⋅log2​(n+1) 공식
	- RT(반응시간) = 사용자가 결정하는 시간
	- n = 선택 가능한 옵션의 개수
	- a, b = 경험적 상수(문맥과 시스템이 따라 다름) (ex : a는 준비하고 반응하는 시간 b는 선택지 증가에 따른 시간 추가)
	- '정보 이론'의 개념과도 연관됨, 인간이 정보를 처리하는 과정에서 발생하는 인지적 부담을 수량화하는 데에 사용
	- UX/UI디자인에서 선택 과부화 문제를 해결하는 중요한 이론적 근거가 됨

힉의 법칙과 UX디자인의 상관관계

	- 힉의 법칙은 UX/UI디자인에서 인지적 부담을 줄이는 데 매우 중요한 역할을 함
	- 사용자는 많은 선택지가 주여질 경우 결정 피로를 경험하며 이는 서비스 이탈률 증가로 이어짐

힉의 법칙이 적용되는 주요 영역

	- 네비게이션 설계 : 직관적인 정보 계층 구조를 제공하여 사용자의 탐색 부담 줄임
	- CTA(Call-To-Action) 버튼 설계 : 핵심적인 행동을 유도해 사용자의 결정을 단순화함
	- 폼 디자인 : 불필요한 입력 필드를 최소화해 사용자 경험 개선
	- 이커머스 및 제품 필터링 : 필터 옵션을 최적화해 사용자의 검색 시간을 단축

인지 부하 이론과의 연계

	- 인지 부하 이론 : 인간의 작업 기억은 한정된 용량을 가지므로 과도한 정보가 주어질 경우 정보 처리 능력이 저하됨
	- 따라서 UX 설계는 사용자가 한번에 처리해야할 정보량을 적절히 조절해야함

힉의 법칙 활용 팁

	- 선택지 최소화 : 한번에 고려할 선택지를 줄이는 정보 처리 속도 업 ( ex : 관련 상품 그룹화 필터링 제공으로 줄임)

	- 선택지 계층화 : 메뉴 구조와 네비게이션을 논리적으로 계층화 하면 사용자의 탐색 경험을 최적화 가능
		- ex : 대시보드에 주요 메뉴를 상위 배치 후 세부 옵션은 드롭다운으로 제공

	-  점진적 정보 제공 : 사용자가 처음부터 모든 정보를 받는다면 인지적 과부화 발생 가능성 큼 
		- 따라서 필요한 정보만 제공 후 나머지는 점진적으로 노출하는 것이 효과적임 
		- ex) 회원가입시 기본 정보만 처음에 입력받고 다음 단계에서 하나씩 받는 방식을 쓰면 이탈률이 줄 수 있음

	- 시각적 계층 구조 활용 : 가장 중요한 선택지를 색깔로 강조하고 버튼 크기등 UI요소의 가독성을 개선해야함

힉의 법칙을 적용한 UX디자인 사례

	- Apple의 제품선택 UI : 사용자에게 기본적으로 두 가지 모델 중 선택하게 하고 추가적인 사양 설정 유도
	- Netflix의 콘텐츠 추천 알고리즘 : 사용자가 너무 많은 콘텐츠를 탐색하지 않도록 개인화된 추천 시스템을 적용
	- Amazon의 원클릭 구매 시스템 : 최소 단계만 거쳐 결제가 완료되도록 설계함
	- 구글의 검색 자동 완성 : 검색어를 입력할 떄 자동 완성 기능을 제공해 선택지를 제한함으로 검색 과정을 단순화함

모바일 UX와 힉의 법칙 적용

	- 모바일 UI에서 선택지 최소화 : 작은 화면 크기로 인해 선택지 최소화가 중요, 대표적인 방법으로 탭 구조, 아코디언 메뉴등이 있음
	- 반응 속도 고려한 UX 최적화 : 인터페이스끼리의 반응 속도가 중요함, 불필요한 애니메이션을 줄이고 가벼운 인터페이스 유지

결론적인 힉의 법칙

	- 힉의 법칙은 사용자의 선택 과정을 최적화 하고 불필요한 인지 부담을 줄이는 핵심적인 원칙
	- 이름 올바르게 적용하면 UX 만족도를 높이고 전환률 개선, 이탈률 감소, 사용자 만족도 증가등의 실질적인 효과를 기대 가능

핵심 활용 방법

	- 선택지 최소화 : 직관적이고 간결한 디자인 적용
	- 점진적 정보 제공 : 한번에 과도한 정보 제공을 하지 않음
	- 시각적 계층 구조 활용 : 핵심 요소를 강조해 사용자의 선택 유도
	- 모바일 UX 최적화 : 터치 기반 환경에 맞게 불필요한 요소 제거
